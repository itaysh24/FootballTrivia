================================================================================
                        NAVIGATION METHODS USED
================================================================================

1. Navigator.push(context, MaterialPageRoute(...))
   - HomePage → GameScreen
   - RoadToGloryScreen → League Map screens
   - League Maps → GameScreen (RTG mode)

2. Navigator.pushNamed(context, 'route')
   - GameModesPage → RoadToGloryScreen
   - GameModesPage → TrainingScreen (via /training route)

3. Navigator.pushReplacementNamed(context, 'route')
   - TutorialScreen → MyHomePage (/)
   - MyHomePage → TutorialScreen (/tutorial)

4. GameModeNavigator helper methods
   - startCasualMode(context)
   - startRushMode(context)
   - startRoadToGloryMode(context, level, leagueId)

5. Navigator.pop()
   - All screens with back buttons
   - Dialog dismissals
